**Zen Spotlight Search** - dim and blur the page behind your search bar for a calmer, focused experience.

## Features

- Enable or disable the spotlight effect separately for:
  - Floating search bar
  - Docked address bar

- Adjustable filter settings:
  - Blur radius
  - Brightness level
  - Transition speed
